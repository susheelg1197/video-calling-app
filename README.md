# Video Calling (Zoom Clone)
A nodejs, socket.io based video calling application like zoom.

## Deployment

This app is now deployed in Heroku.

### Steps to deploy
1. ```
    Install Heroku CLI 
    sudo snap install --classic heroku

    ```
2. ```
    heroku login
    ```
3. ```
    heroku create
    ```
4. ```
    connect with git 
    git add .
    git commit -m
    git push
    ```
